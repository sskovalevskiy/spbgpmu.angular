What is a theme?

A theme, template, skin and etc., is a ready to use design and layout you can use on your existing WordPress site. WordPress will handle all the contents for you (such as posts, pages, comments), while the themes transforms those contents into a beautiful and clean layout for your website visitors/customers. Many themes come with quite a bit of customizable options, but all of our themes run on the Woo Framework which built with lots of great features and goodies. Simply take a tour of our framework, or do the test drive by yourself on our themes playground/showcase.

What is WordPress ecommerce theme?

WordPress e-Commerce is an e-Commerce plugin for WordPress. Since WordPress has now expanded to the e-Commerce business, all themes we provided are supported by WooCommerce and WordPress e-Commerce from GetShopped.org. To see available themes, you can go to WooCommerce Themes and WordPress e-Commerce Themes.

In which browsers and browser versions did you test the themes?

We tested all of our themes on Internet Explorer 8, the latest versions of Firefox, Safari, and Chrome. Regarding to operating systems, our themes are independent from the operating system you use. It will work as long as WordPress is installed correctly. We’re not supported to the Internet Explorer 6 and 7. If you use Internet Explorer 6 or 7, please consider to upgrade it to the latest Internet Explorer, or you can simply switch to a different browser to have the best experience with our themes.

Do your themes work with plugins?

Yes, our themes work with plugins and most plugins should work with our themes, HOWEVER, we can’t guarantee they all will. There are some plugins that conflicts with code used in the themes, which will make our themes doesn&#8217;t function properly. There are so many plugins available, and if the plugin has been coded to WordPress standards, chances are 99% it works well with our themes. If it doesn&#8217;t, we’re happy to troubleshoot and figure out which one is being the problem. Just drop us a message on the contact page.
